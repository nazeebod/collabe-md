# Коллаборация

Совместное редактирование Markdown-документа в реальном времени несколькими участниками по одной ссылке.

## Назначение

Несколько человек могут одновременно редактировать один документ. Изменения текста появляются у всех участников без ручного обновления страницы. Каждый участник видит, кто ещё находится в документе.

## Как присоединиться

Достаточно открыть ссылку на документ. Регистрация и вход не требуются. Каждому новому участнику автоматически присваивается случайное имя животного (Fox, Bear, Otter и др.) и свой цвет.

## Присутствие участников

В шапке документа отображается блок **Online:** со списком имён активных участников. Каждое имя показано на цветной плашке, соответствующей цвету участника в редакторе.

Список обновляется при подключении и отключении участников.

## Совместное редактирование

- Все изменения в тексте синхронизируются между открытыми вкладками и браузерами.
- Предпросмотр справа обновляется по мере ввода любым участником.
- В редакторе видны цветные курсоры других участников с их именами.
- При обрыве соединения показывается уведомление о переподключении; после восстановления связи редактирование продолжается с актуальным содержимым.

## Сохранение

Содержимое документа сохраняется на сервере автоматически. При повторном открытии ссылки текст восстанавливается.

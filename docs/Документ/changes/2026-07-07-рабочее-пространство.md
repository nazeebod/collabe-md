# Рабочее пространство документа

## Описание изменения

Реализована страница документа `/d/:documentId`: загрузка метаданных, редактируемый заголовок, копирование ссылки, split-view с редактором Markdown и живым предпросмотром.

## Структура данных

### DocumentMeta (API)

```typescript
type DocumentMeta = {
  id: string;
  title: string;
  createdAt: string;  // ISO 8601
  updatedAt: string;  // ISO 8601
};
```

### Yjs-документ

- Комната = `documentId` (UUID).
- Текстовое поле `content` (`Y.Text`) — тело Markdown-документа.
- Состояние персистится в `documents.yjs_state` (bytea) через Hocuspocus.

### REST API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/documents/:id` | Метаданные документа |
| PATCH | `/api/documents/:id` | Обновление заголовка `{ title: string }` |

Валидация `id`: UUID v4 (`isValidUuid`). Ошибки: 400 (невалидный id / пустой title), 404 (не найден).

## Интерфейс

### Шапка (`document-header`)

- Ссылка «Collabe MD» → `/`.
- `<input>` заголовка (`data-testid="document-title"`, `aria-label="Document title"`).
- `PresenceBar` — список онлайн-участников (см. раздел «Коллаборация»).
- Кнопка **Copy link** (`data-testid="copy-link"`).
- Сообщение «Link copied» (2 сек).

### Рабочая область

- `SplitLayout` — левая панель 60% по умолчанию, перетаскиваемый разделитель (20–80%).
- Левая панель: `MarkdownEditor` (CodeMirror 6 + y-codemirror).
- Правая панель: `MarkdownPreview` (react-markdown + remark-gfm).

### Состояния загрузки

- Ошибка загрузки: текст ошибки + ссылка на главную.
- Ожидание коллаборации: «Loading editor...».
- Отключение WS: баннер `data-testid="disconnect-banner"`.

## Бизнес-логика и валидация

1. При монтировании: `GET /api/documents/:id` → установка `title`, добавление в недавние.
2. `onBlur` заголовка: `PATCH` только если `title.trim()` не пуст; при ошибке локальный заголовок сохраняется.
3. `handleCopyLink`: `navigator.clipboard.writeText(window.location.href)`.
4. Контент редактора синхронизируется через Yjs; `useCollaboration` подписывается на `ytext.observe` для обновления preview.
5. Предпросмотр: пустой контент → placeholder `*Start writing markdown...*`.

## Права доступа (RBAC)

Доступ по capability URL (UUID v4). Аутентификация отсутствует — любой с ссылкой может читать и редактировать.

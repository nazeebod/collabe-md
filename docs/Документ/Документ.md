# Документ

Рабочее пространство для редактирования одного Markdown-документа. Открывается по уникальной ссылке `/d/:id`.

## Назначение

Пользователь работает с содержимым документа, задаёт заголовок, копирует ссылку для приглашения соавторов и видит результат разметки в режиме предпросмотра.

## Заголовок и навигация

- В шапке отображается ссылка **Collabe MD** — возврат на главную.
- Редактируемое поле заголовка документа. По умолчанию «Untitled».
- При потере фокуса поля заголовок сохраняется, если он не пустой.
- Если документ не найден, показывается сообщение об ошибке и ссылка «Back to home».

## Копирование ссылки

Кнопка **Copy link** копирует текущий адрес страницы в буфер обмена. На две секунды появляется подтверждение «Link copied».

## Рабочая область

Экран разделён на две панели:

- **Слева** — редактор Markdown с подсветкой синтаксиса и нумерацией строк.
- **Справа** — живой предпросмотр отформатированного текста (заголовки, списки, таблицы, ссылки и т.д.).

Между панелями — перетаскиваемый разделитель: пользователь может изменить ширину панелей (от 20% до 80% для левой). При пустом содержимом в предпросмотре показывается подсказка «Start writing markdown...».

## Состояние подключения

При потере соединения с сервером в верхней части рабочей области появляется баннер «Connection lost. Reconnecting...». После восстановления связи баннер исчезает.

# Главная

Стартовая страница сервиса Collabe MD — точка входа для создания новых документов и возврата к недавно открытым.

## Назначение

Пользователь попадает на главную при открытии корневого адреса сервиса. Здесь можно быстро создать новый документ и перейти к ранее открытым файлам без регистрации и входа в аккаунт.

## Стартовый экран

На странице отображается название сервиса, краткое описание возможностей и кнопка **Create document** («Создать документ»).

При нажатии на кнопку создаётся новый документ с заголовком по умолчанию «Untitled». Пользователь сразу переходит в рабочее пространство документа. Пока документ создаётся, кнопка недоступна и показывает состояние загрузки. При ошибке создания отображается сообщение об ошибке.

## Недавние документы

Под блоком создания документа может отображаться список **Your recent documents** («Ваши недавние документы»).

- Список формируется из документов, которые пользователь открывал или создавал в этом браузере.
- Для каждого пункта показываются заголовок и дата последнего открытия.
- Нажатие на пункт открывает соответствующий документ.
- Рядом с каждым пунктом есть кнопка удаления — документ убирается только из локального списка, сам файл на сервере не удаляется.
- Если недавних документов нет, блок не отображается.
- Данные хранятся только в браузере пользователя и не синхронизируются между устройствами.

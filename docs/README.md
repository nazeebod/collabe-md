# Документация Collabe MD

Функциональные спецификации и история изменений сервиса совместного редактирования Markdown-документов.

## Структура

```
docs/
  README.md                          # этот файл — оглавление
  Главная/
    Главная.md                       # стартовый экран, создание документа, недавние документы
    changes/
      2026-07-07-стартовый-экран.md
      2026-07-10-недавние-документы.md
  Документ/
    Документ.md                      # рабочее пространство документа
    changes/
      2026-07-07-рабочее-пространство.md
  Коллаборация/
    Коллаборация.md                  # совместное редактирование в реальном времени
    changes/
      2026-07-07-realtime-редактирование.md
  general/
    changes/
      2026-07-10-плашка-github.md    # сквозные изменения без привязки к разделу
```

## Разделы

| Раздел | Описание |
|--------|----------|
| [Главная](Главная/Главная.md) | Стартовая страница: создание документа и список недавно открытых |
| [Документ](Документ/Документ.md) | Рабочее пространство: заголовок, копирование ссылки, редактор и предпросмотр |
| [Коллаборация](Коллаборация/Коллаборация.md) | Совместное редактирование, присутствие участников, курсоры |

## Процесс разработки

Новый функционал описывается в change-файле, согласуется, затем обновляется главное ТЗ раздела и реализуется в коде. Подробности — в [`.claude/skills/development-workflow/SKILL.md`](../.claude/skills/development-workflow/SKILL.md).
